# Student Management System

A web-based Student Management System built primarily with Python, HTML, and CSS. This project provides tools to efficiently manage student data, including registration, grading, and record keeping, all through an easy-to-use web interface.

## Features

- Add, edit, and delete student records
- Manage courses and enrollments
- Record and view grades
- Search and filter student information
- Responsive user interface
